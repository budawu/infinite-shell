# PI Shell-基于Python的命令行

### 事先声明：这只是一个初中小白瞎搞的东西，不喜勿喷！
## 实现原理
字符串切片，关键词识别  
## 语法
关键词+空格+关键词
截止0.0.3版本，支持的指令为：
`cd` `$`(运行系统默认shell指令)
`echo`(支持字符串)`runpy`(运行Python代码)  
`exit`
e.g:

```shell
C:/pythondemo/pish π  echo "hello,pi shell!"
```
# 不成熟的指令
 * var (变量)
 * 算式
 * 按enter键会自动退出
